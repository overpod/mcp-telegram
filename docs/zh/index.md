## 使用示例

连接后，直接用自然语言与 AI 助手对话：

> **你：** 总结我 Telegram 的未读消息

> **你：** 在所有聊天中搜索关于"项目截止日期"的消息

> **你：** 发送"下午3点开会"到工程师群

> **你：** 我安装了哪些贴纸包？

AI 助手在后台使用 MCP Telegram 工具 — 无需记忆任何命令。

## 不想自建？

使用**[云版本](https://mcp-telegram.com)** — 30 秒内通过二维码将 Telegram 连接到 Claude.ai 或 ChatGPT。无需 API 密钥、终端或配置。
