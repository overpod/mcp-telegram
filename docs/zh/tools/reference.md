# 工具参考

MCP Telegram 提供 **59 个工具**，按类别组织。所有工具通过 MCP 自动发现。

## 认证 (2)

| 工具 | 说明 |
|------|------|
| `telegram-status` | 检查连接状态和账户信息 |
| `telegram-login` | 生成认证二维码 |

| 工具 | 说明 |
|------|------|
| `telegram-send-message` | 发送消息到任何聊天 |
