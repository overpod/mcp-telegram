# 贴纸

发送贴纸、浏览已安装的贴纸包、搜索新贴纸。

## 发送贴纸

> **你：** 给 @alice 发一个竖大拇指的贴纸

AI 会在您已安装的贴纸中搜索并发送最匹配的。

## 浏览贴纸

> **你：** 我安装了哪些贴纸包？

> **你：** 显示我最近使用的贴纸

## 浏览贴纸包

> **你：** 显示 "Pepe" 贴纸包的贴纸

## 搜索贴纸包

> **你：** 搜索有猫的贴纸包

## 可用工具

| 工具 | 功能 |
|------|------|
| `telegram-send-sticker` | 发送贴纸 |
| `telegram-get-installed-stickers` | 已安装贴纸包列表 |
| `telegram-get-recent-stickers` | 最近使用的贴纸 |
| `telegram-get-sticker-set` | 浏览贴纸包 |
| `telegram-search-sticker-sets` | 搜索贴纸包 |
