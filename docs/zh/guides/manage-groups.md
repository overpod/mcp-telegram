# 管理群组

MCP Telegram 提供完整的群组管理 — 创建、配置、邀请成员、设置管理员等。

## 创建群组

> **你：** 创建一个叫"项目 Alpha"的 Telegram 群，邀请 @alice 和 @bob

## 邀请成员

> **你：** 邀请 @charlie 加入项目 Alpha 群

## 设置管理员

> **你：** 让 @alice 成为项目 Alpha 的管理员

## 管理

> **你：** 将 @spammer 踢出群组

> **你：** 在工程师群中封禁 @troll

## 群组设置

> **你：** 在项目 Alpha 群设置一周后自动删除消息

> **你：** 将 Random 群静音

## 可用工具

| 工具 | 功能 |
|------|------|
| `telegram-create-group` | 创建群组 |
| `telegram-edit-group` | 编辑标题、描述、头像 |
| `telegram-invite-to-group` | 邀请用户 |
| `telegram-join-chat` | 通过邀请链接加入 |
| `telegram-leave-group` | 离开群组 |
| `telegram-kick-user` / `telegram-ban-user` | 踢出/封禁 |
| `telegram-set-admin` / `telegram-remove-admin` | 管理管理员 |
| `telegram-get-my-role` | 查看自己的角色 |
| `telegram-get-chat-members` | 成员列表 |
| `telegram-create-invite-link` | 创建邀请链接 |
| `telegram-mute-chat` | 静音通知 |
| `telegram-set-auto-delete` | 自动删除计时器 |
