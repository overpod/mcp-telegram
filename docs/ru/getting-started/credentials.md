# Получение API-ключей Telegram

Перед использованием MCP Telegram вам нужны **API ID** и **API Hash** от Telegram. Это бесплатно и занимает около 2 минут.

## Пошаговая инструкция

### 1. Откройте my.telegram.org

Перейдите на [my.telegram.org](https://my.telegram.org) и войдите с номером телефона (с кодом страны, например `+79001234567`).

### 2. Перейдите в API Development Tools

После входа нажмите **«API development tools»**.

### 3. Создайте приложение

Заполните форму:
- **App title** — любое (например «My MCP»)
- **Short name** — любое (например «mymcp»)
- **Platform** — любая (не имеет значения)
- **Description** — оставьте пустым или напишите что угодно

Нажмите **Create application**.

### 4. Скопируйте ключи

Вы увидите два значения:
- **App api_id** — число, например `12345678`
- **App api_hash** — строка, например `a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4`

Сохраните их — они понадобятся для установки и входа.

::: tip
Эти ключи идентифицируют ваше приложение для Telegram. Это **не** логин — аутентификация делается отдельно через QR-код.
:::

::: warning
Храните API-ключи в секрете. Не коммитьте в публичные репозитории. Используйте переменные окружения.
:::
