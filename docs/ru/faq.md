# FAQ

## Это бот?

Нет. MCP Telegram работает как **юзербот** — под вашим личным аккаунтом Telegram, а не как бот. Это значит, что у него есть доступ ко всем вашим чатам, контактам и истории, как у приложения Telegram на телефоне.

## Это безопасно?

Да. Все данные идут напрямую с вашей машины на серверы Telegram через протокол MTProto. Никакие данные не проходят через сторонние сервисы. Файл сессии хранится локально с ограниченными правами (0600).

## Мой аккаунт могут забанить?

MCP Telegram использует официальный API Telegram (MTProto) через GramJS. Пока вы не злоупотребляете (массовый спам, автоматический флуд, скрейпинг), с аккаунтом всё будет в порядке. Используйте ответственно.

## Нужно ли держать терминал открытым?

Нет. MCP Telegram работает как часть вашего MCP-клиента (Claude Desktop, Cursor и т.д.). Когда клиент открыт — сервер работает. Закрыли — сервер остановился.

## Можно использовать несколько аккаунтов?

Да! См. руководство [Несколько аккаунтов](/ru/guides/multiple-accounts).

## Чем отличается самостоятельная установка от облака?

| | Самостоятельная | Облако ([mcp-telegram.com](https://mcp-telegram.com)) |
|---|---|---|
| **Настройка** | API-ключи, установка, QR-вход | Только QR-код |
| **Где работает** | Ваша машина | Наши серверы |
| **Стоимость** | Бесплатно (open source) | Бесплатный тариф + платные |
| **Клиенты** | Любой MCP-клиент | Claude.ai, ChatGPT |
| **Приватность** | Полный контроль | Сообщения обрабатываются на наших серверах |

## Работает с каналами?

Да. Можно читать сообщения каналов, искать по ним, а если вы админ — отправлять сообщения и управлять каналом.

## Поддерживаются форумы (топики)?

Да. Полная поддержка — список топиков, чтение по топикам, отправка в конкретные топики, счётчики непрочитанных по топикам.

## Сколько инструментов?

**59 инструментов** в версии v1.24.0. См. [полный справочник](/ru/tools/reference).

## Можно без Node.js?

Да! Скачайте [готовые бинарники](https://github.com/overpod/mcp-telegram/releases) — самостоятельные исполняемые файлы для Linux, macOS и Windows.
