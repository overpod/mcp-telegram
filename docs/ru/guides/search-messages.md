# Поиск сообщений

MCP Telegram даёт мощный поиск по чатам — находите любое сообщение по ключевым словам.

## Поиск в конкретном чате

> **Вы:** Найди «заметки со встречи» в группе Инженеры

Использует `telegram-search-messages` с серверным поиском Telegram (быстрый, работает в чатах с миллионами сообщений).

## Поиск по всем чатам

> **Вы:** Найди во всех моих чатах Telegram сообщения про «счёт»

Использует `telegram-search-global` для поиска по всем чатам сразу.

## Поиск чатов

> **Вы:** Найди все мои группы Telegram про AI

Использует `telegram-search-chats` для поиска по названию чата.

## Доступные инструменты поиска

| Инструмент | Что делает |
|------------|-----------|
| `telegram-search-messages` | Поиск в конкретном чате по ключевым словам |
| `telegram-search-global` | Поиск по всем чатам сразу |
| `telegram-search-chats` | Поиск чатов по названию |

## Советы

- Поиск серверный — быстрый даже для больших чатов
- Результаты включают имя отправителя, дату и текст
- Поддерживается пагинация для больших наборов результатов
