# Управление группами

MCP Telegram даёт полное управление группами — создание, настройка, приглашение участников, назначение администраторов и многое другое.

## Создать группу

> **Вы:** Создай группу Telegram «Проект Альфа» и пригласи @alice и @bob

## Пригласить участников

> **Вы:** Пригласи @charlie в группу Проект Альфа

## Назначить администратора

> **Вы:** Сделай @alice админом в Проекте Альфа

## Модерация

> **Вы:** Кикни @spammer из группы

> **Вы:** Забань @troll в группе Инженеры

## Настройки группы

> **Вы:** Установи автоудаление через неделю в группе Проект Альфа

> **Вы:** Замьють группу Random

## Доступные инструменты

| Инструмент | Что делает |
|------------|-----------|
| `telegram-create-group` | Создать группу |
| `telegram-edit-group` | Изменить название, описание, фото |
| `telegram-invite-to-group` | Пригласить пользователей |
| `telegram-join-chat` | Вступить по инвайт-ссылке |
| `telegram-leave-group` | Покинуть группу |
| `telegram-kick-user` / `telegram-ban-user` | Кикнуть / забанить |
| `telegram-set-admin` / `telegram-remove-admin` | Управление админами |
| `telegram-get-my-role` | Проверить свою роль |
| `telegram-get-chat-members` | Список участников |
| `telegram-create-invite-link` | Создать инвайт-ссылку |
| `telegram-mute-chat` | Замьютить уведомления |
| `telegram-set-auto-delete` | Таймер автоудаления |
