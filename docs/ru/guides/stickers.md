# Стикеры

Отправляйте стикеры, просматривайте установленные паки и ищите новые.

## Отправить стикер

> **Вы:** Отправь стикер с большим пальцем вверх @alice

ИИ найдёт подходящий стикер в ваших установленных паках и отправит.

## Поиск стикерпаков

> **Вы:** Найди стикерпаки с котами

## Доступные инструменты

| Инструмент | Что делает |
|------------|-----------|
| `telegram-send-sticker` | Отправить стикер в чат |
| `telegram-get-installed-stickers` | Список установленных паков |
| `telegram-get-recent-stickers` | Недавно использованные стикеры |
| `telegram-get-sticker-set` | Просмотр стикеров в паке |
| `telegram-search-sticker-sets` | Поиск стикерпаков |
