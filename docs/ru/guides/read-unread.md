# Непрочитанные сообщения

Один из самых популярных сценариев — пусть ИИ-ассистент расскажет, что вы пропустили.

## Просто спросите

> **Вы:** Какие у меня непрочитанные сообщения в Telegram?

ИИ использует `telegram-get-unread` для получения чатов с непрочитанными, затем `telegram-read-messages` для чтения содержимого.

## Расширенный вариант

> **Вы:** Проверь непрочитанные в Telegram. Рабочие чаты — подробно, личные — кратко.

## Отметить как прочитанное

> **Вы:** Отметь группу Инженеры как прочитанную

## Как это работает

За кулисами ИИ использует эти инструменты:

1. **`telegram-get-unread`** — список чатов с непрочитанными сообщениями
2. **`telegram-read-messages`** — чтение сообщений из каждого чата
3. **`telegram-mark-as-read`** — опционально отмечает как прочитанное

## Советы

- Можно запросить конкретные чаты: *«Непрочитанные от команды дизайна»*
- Работает с группами, каналами и личными сообщениями
- Поддерживаются топики форумов: *«Что нового в топике #general?»*
